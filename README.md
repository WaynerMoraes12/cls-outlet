# 🛒 CLS Outlet — Site Institucional

Site institucional da **CLS Outlet**, empresa especializada em venda de produtos variados com os melhores preços da região, com unidades em **Araras – SP** e **Santa Bárbara d'Oeste – SP**.

---

## 📋 Sobre o Projeto

Site desenvolvido em **HTML, CSS e JavaScript puro** (vanilla), sem dependências externas ou frameworks. Uma única página (single page) responsiva, otimizada para desktop, tablet e celular.

---

## 🗂️ Estrutura de Arquivos

```
cls-outlet/
├── index.html          → Página principal do site
├── foto_placa.jpg      → Foto da placa da loja (usada no Hero)
├── foto_entrada.jpg    → Foto da entrada da loja (usada na seção Sobre)
├── foto_fachada.jpg    → Foto interna da loja (usada na Galeria)
├── foto_fachada_2.jpg  → Foto interna com motos (usada na Galeria)
├── foto_fachada_3.jpg  → Foto interna com produtos (usada na Galeria)
└── README.md           → Este arquivo
```

---

## 🚀 Como Rodar Localmente

### Opção 1 — Live Server (VS Code)
1. Instale a extensão **Live Server** de Ritwick Dey no VS Code
2. Clique com botão direito no `index.html`
3. Selecione **"Open with Live Server"**
4. O site abre automaticamente no navegador

### Opção 2 — Abrir direto no navegador
1. Navegue até a pasta `cls-outlet/`
2. Dê dois cliques no arquivo `index.html`
3. Ele abre direto no seu navegador padrão

## 🛠️ Tecnologias Utilizadas

- **HTML5** — estrutura semântica da página

## 📱 Responsividade

O site é totalmente responsivo com breakpoints para:
- **Desktop** — acima de 1024px
- **Tablet** — até 1024px
- **Mobile** — até 768px e 480px
